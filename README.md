# AskMate 
Q and A Forum software written in Python Flask and HTML / CSS
How to use:
Forum uses PostgreSQL (ver 12 at least)
use my dump (located in the sample_data folder)
put your postgres password user.txt ( you can use environmental variable to hide your credentials (need to rewrite the data_manager.py)
