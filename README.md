AskMate ver 2
Q & A website written in Python + Flask + Jinja
If u want to use this website you need use my PostgreSQL dump (which is located in the sample_data folder ) and import it to your PostGreSQL- server.
Don't forget to write your credentials to the database_common.py, data_manager.py and user.txt(only need your SQL password) file otherwise the program won't work.
